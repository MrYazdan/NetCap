## Use Cases

1. **Debugging**: Inspect HTTP requests/responses during development
2. **API Testing**: Capture and analyze API calls
3. **Security Auditing**: Review network traffic for vulnerabilities
4. **Performance Monitoring**: Track request/response times
5. **Regression Testing**: Replay captured requests
6. **Learning**: Understand how applications make network calls
